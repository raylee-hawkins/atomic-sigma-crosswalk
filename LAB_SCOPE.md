# Lab Scope

This lab is intentionally narrow and public-safe.

## In Scope

- One technique only.
- One Atomic candidate only.
- One Sigma rule.
- One Splunk SPL translation.
- One Microsoft Sentinel KQL translation.
- One validation note.

## Current Boundary

Pending collaborator-provided Atomic Red Team candidate and telemetry context.

No ATT&CK ID, Atomic test details, logsource, event identifiers, query logic, false positives, severity, or validation outcome are asserted in this scaffold.

## Out of Scope

- Production deployment.
- SOC coverage claims.
- Enterprise detection coverage.
- Broad detection library development.
- Multiple techniques or multiple Atomic candidates.
- Private, customer, or sensitive telemetry.
- Live validation claims without recorded evidence.
- Guessing detection logic or telemetry fields.
