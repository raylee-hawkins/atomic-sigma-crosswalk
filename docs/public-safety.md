# Public Safety

This repo is public and must not contain:

* private data
* customer data
* raw sensitive telemetry
* lab IP addresses unless intentionally approved
* secrets
* credentials
* real internal hostnames
* unredacted screenshots
* exploit instructions beyond benign detection context

## Claim Rules

Allowed current claim:
A public personal GitHub repo exists for a public-safe detection translation collaboration.

Not allowed yet:

* detection completed
* validation completed
* Atomic test executed by this repo
* Splunk detection works
* Sentinel detection works
* Sigma rule is correct
* production readiness
* SOC deployment
* enterprise coverage
